🏗 Architecture Overview

▼

The system is a full-stack, multi-platform HR solution consisting of 3 applications connected via REST API and Socket.IO for real-time communication.

Key Architecture Decisions

🔌 Generic CRUD Pattern

Single dynamic route handles all CRUD for 60+ collections — no duplicate route/controller pairs.

⚡ Real-Time Broadcasting

Any data mutation triggers Socket.IO dataUpdate to all connected clients.

🕐 Server-Side Timestamps

All time operations use server time in Asia/Phnom_Penh — prevents clock manipulation.

📍 Haversine GPS Verification

Check-in only valid if employee is within 100m of a registered location.

🔒 Progressive Account Lockout

5 failed attempts → exponential lockout: 15min, 30min, 60min.

🌐 Bilingual Support

All entities have dual fields: nameKh / nameEn. 312 translation keys in mobile.

🔒 Security Features

▼

🔑 JWT Authentication

Stateless token-based authentication with expiration. Used across all three platforms.

🔐 bcrypt Password Hashing

Passwords hashed with salt. Never stored in plaintext.

🚫 Progressive Account Lockout

5 failed attempts trigger exponential lockout: Lock 1 = 15min, Lock 2 = 30min, Lock 3 = 60min. Formula: 15 × 2(count-5)

🔏 OTP Email Verification

6-digit code sent via email, expires in 5 minutes. Used for login verification and account recovery.

📱 Device Fingerprinting

Single-device enforcement — users can only log in from one registered device.

📷 Biometric Auth (Mobile)

Fingerprint and Face ID support via local_auth on Flutter mobile app.

🛡 Helmet.js Headers

HTTP security headers to protect against common web vulnerabilities.

⚠ Rate Limiting

Request throttling via express-rate-limit to prevent DDoS and brute-force attacks.

🕐 Server-Side Timestamps

All time operations use server time (Asia/Phnom_Penh) — prevents clock manipulation fraud.

📍 GPS Geofence Verification

Haversine formula calculates distance. Check-in only valid within 100m of registered location.

📄 Rotating File Logs

Daily log rotation with gzip compression. 14-day retention for audit trails.

🔑 AES Encryption

Custom crypto operations for sensitive data encryption/decryption.
